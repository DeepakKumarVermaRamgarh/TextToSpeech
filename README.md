
# Text To Speech Converter

This is a simple text-to-speech converter web application using HTML, CSS, and JavaScript.




## Getting Started

Clone the repository or download the project files.
Open the index.html file in a web browser.

```bash
  git clone https://github.com/DeepakKumarVermaRamgarh/TextToSpeech
```
    
## Features

The "Text To Speech Converter" allows users to enter text in a textarea and convert it to speech. The converted speech can be played back using the available voices on the user's device.

## Usage/Examples

- Upon opening the web page, the user interface displays a heading "Text To Speech Converter", a textarea for entering text, a select dropdown for choosing the voice, and a button for initiating the text-to-speech conversion.
- Select the desired voice from the available options in the voice select dropdown.
- Enter the text you want to convert into speech in the textarea.
- Click the "Listen" button to start the text-to-speech conversion. The converted speech will be played back using the selected voice.
Customize the CSS in the style.css file to modify the appearance of the application if desired.
